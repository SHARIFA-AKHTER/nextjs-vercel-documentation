# Setting up a GitHub Repository

To manage our code and enable deployment, we need to host our project on GitHub.

### Step 1: Create a Repository
1. Go to [GitHub](https://github.com/SHARIFA-AKHTER/nextjs-vercel-documentation).
2. Name your repository (e.g., `nextjs-vercel-documentation`).
3. Keep it **Public** and click **Create repository**.

### Step 2: Push Your Code
Run these commands in your project folder to link your local code to GitHub:
`git init`
`git add .`
`git commit -m "Initial commit"`
`git branch -M main`
`git remote add origin [https://github.com/SHARIFA-AKHTER/nextjs-vercel-documentation.git]`
`git push -u origin main`
